У тебя есть список полных имём участников беседы и их никнеймы из Google Meet в скобках (если есть), которые удалось определить с помощью аудио транскрибатора:
[CONVERSATION_MEMBERS]

Так как транскрибатор работает не точно, а в беседе не обязательно будет упомянаться полное имя участника, эти имена являтся приблизительными.

Также у тебя есть список данных сотрудников компании:
[COMPANY MEMBERS]

Твоя задача попытаться определить, является ли участник беседы сотрудником компании с помощью предоставленного списка содруников компании, и, если является, написать его полное имя корректно, без ошибок транскрибации.

Оформи ответ в json формате в следующем виде:
{
    "speaker_0": {
        "is_company_member": true,
        "full_name": "Долгов Данил Петрович"
    },

    "speaker_1": {
        "is_company_member": false,
        "full_name": ""
    },
    "speaker_2": {
        "is_company_member": true,
        "full_name": "Зимин Евгений Сергеевич"
    }
}

ПРИМЕЧАНИЕ:
1. При написании ответа тебе не нужно писать ```json и закрывать ```, чтобы форматировать ответ.
2. Если тебе известно только имя или другая небольшая часть полного имени (например только отчество, ну ты понял), которая не позволяет точно иденцифицировать пользователя (только по имени нельзя определить является ли человек сотрудником компании, ведь очень много людей может носить одно и то же имя), тогда лучше укажи "is_company_member": false для этого сотрудника - так будет в разы достовернее.
3. Если тебе известно только имя и никнейм из Google Meet, тогда старайся пополной использовать всю данную тебе информацию, чтобы иденцифицировать пользователя. В редких случая это бывает возможно, ведь никнеймы часто могут быть похожи на название почты пользователя.
